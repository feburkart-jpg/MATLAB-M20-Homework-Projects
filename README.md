# Simulation of Microstructure Evolution and Grain Growth Using Cellular Automata in MATLAB
The attatched files summarize my final project at UCLA's introductory MATLAB course, designed for engineering majors. The project demonstrates my competency with fundemental MATLAB concepts, and s my ability to apply them to a cohesive project. 

[Final Project](Burkart_Final_Project_Interactive.m):
This project implements a 2D simulation of microstructure evolution during annealing using MATLAB and a lattice Potts model with a Monte Carlo (Metropolis) approach. The simulation models grain growth by iteratively selecting random cells and evaluating orientation changes based on local grain boundary energy and temperature-dependent probabilistic acceptance. A graphical user interface allows users to control parameters such as sample size, number of grain orientations, and temperature, while the output displays a real-time visualization of grain evolution alongside a plot of average grain area versus simulation steps. The project demonstrates how stochastic energy minimization can reproduce realistic grain growth behavior and highlights the impact of temperature on microstructural evolution.
